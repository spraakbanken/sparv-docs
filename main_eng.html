Korp's Annotation Laboratory is mostly an interface to the corpus import pipeline. It can be used to annotate shorter texts using the same tools as are applied in the annotation of the corpora in Korp.

Click here to get to the Annotation Laboratory.

The documentation to the web service (API) belonging to the Annotation laboratory can be found here.
